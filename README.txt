كيفية رفع الملفات للحصول على رابط URL:
1. ارفع مجلد ZIP على أي خدمة مجانية مثل Neocities (https://neocities.org/) أو GitHub Pages.
2. بعد الرفع، ستحصل على رابط URL مباشر للملف index.html.
3. استخدم الرابط في أي أداة تحويل HTML إلى APK (مثل Web2Apk) لتحصل على APK جاهز.
ملاحظات:
- التطبيق يستخدم LocalStorage لتخزين البيانات ولن يرسل أي شيء للسحابة.
- يمكنك فتح التطبيق والعمل عليه دون إنترنت بعد التحويل.
